# Flow Builder

A lightweight, extensible node-based pipeline builder built with Vite, React, and TypeScript.

This project allows users to visually construct data or logic pipelines using draggable nodes and dynamic variable bindings. Nodes can be connected, configured, and composed to form structured execution flows.

The system is designed to be modular, predictable, and scalable — making it suitable for workflow automation, prompt chaining, internal tooling, or experimental graph-based interfaces.
